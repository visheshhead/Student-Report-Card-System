# Student Report Card V2

Console-based Student Report Card System in C with Search & Performance Analytics

---

## Description

This is a menu-driven console application built in C that allows adding student details, automatically calculates GPA including extra-curricular marks, highlights toppers, and provides search & performance analytics.

It simulates a mini academic management system, giving hands-on experience with file handling, arrays, structs, and console-based data visualization.

---
